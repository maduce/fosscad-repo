# Liberator Glock Frame -Proteus

#### Version 
1.0

#### Description
This is a modified Liberator pistol frame designed to accept a Glock barrel instead of the standard Liberator barrel. Print out all Liberator parts except barrel and frame. Also print 4 frame pins.
Print out modified frame included here and purchase a glock barrel. Insert barrel into frame and put the 4th frame pin above to help hold it in place. 

If you test this, do if from far away.
